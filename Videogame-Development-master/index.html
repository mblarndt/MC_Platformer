VOID LEAPER

Void Leaper is a short prototype developed in C/C++ & SDL Libraries made as part of a development subject in CITM-UPC. The game is a platformer that includes moving Entities and functional UI. We hope you enjoy our game.

Controls

  • A/D or LEFT/RIGHT - lateral movement
  • S or DOWN - crouch (allows you to drop from platforms)
  • X or L - Attacking Dash
  • SPACE - Jump. Holding SPACE increases height
  • ESC (in game) - Open Menu

Debug Features

  • F1 - Load 1st Level
  • F2 - Load 2st Level
  • F3 - Alternate Level
  • F5 - Save State
  • F6 - Load State
  • F8 - Toggle GUI debug
  • F9 - Toggle Display Colliders / Pathfinding
  • F10 - Toggle God Mode (while God Mode is on use WASD or ARROWS to fly)
  • F11 - Cap/Uncap FPS to 30
  • ยบ - Open/Close Console
